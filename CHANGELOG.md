# Changelog

All notable changes to Event Budget Tracker are documented here.

## [1.0.0] — 2026-04-01

### Added
- Initial release as part of EventHive open-source toolkit
- Industry-standard cost categories pre-loaded
- Budget / Actual / Variance columns with auto-calculation
- RAG (Red/Amber/Green) status per line item
- Scenario modelling: Best Case, Base Case, Downturn
- PO and invoice tracking with payment status
- Key ratio dashboard (venue %, tech %, staffing %)
- Export to CSV and PDF
